# H1-H6 Word Buddy Quest 單字級數進度

題庫依照 H1 到 H6 統整，不再區分舊版、新版或課別小單元。遊戲會把同一個 H 級別中的單字合併成一個大關卡，並排除英文重複單字。

| 遊戲等級 | 顯示關卡 | 內容 |
| --- | --- | --- |
| Lv.1 | H1 | H1 舊版考卷與新版單字表中不重複單字 |
| Lv.2 | H2 | H2 舊版考卷與新版單字表中不重複單字 |
| Lv.3 | H3 | H3 舊版考卷與新版單字表中不重複單字 |
| Lv.4 | H4 | H4 舊版考卷與新版單字表中不重複單字 |
| Lv.5 | H5 | H5 新版單字表與考卷中不重複單字 |
| Lv.6 | H6 | H6 新版單字表與考卷中不重複單字 |

## 遊戲模式

- 拼字模式：看中文提示，從答案字母加上額外干擾字母的字母池中拼出英文單字。
- 自然發音模式：看/聽單字，選出它包含的字母組合規則。

## 記憶進度

- `localStorage` 會依照玩家名字儲存分數、連續答對、等級、目前模式、選擇的 H 級別、目前題目與下一題位置。
- 同一台裝置可以建立多位玩家，下次打開後選自己的名字即可接續。
- 每位玩家在同一個模式與 H 級別中會有自己的洗牌單字順序。
- 答錯的單字會先放進該級別的複習隊列，等能量要升等時再出現；補答正確後才會真正升等。
- 這份紀錄存在使用中的瀏覽器與裝置上；清除網站資料或換裝置會重新開始。

## 自然發音規則

自然發音模式會從 H1-H6 單字中挑出有明顯字母組合的字，讓孩子練習「看到組合就想到聲音」。

| 類型 | 規則例子 |
| --- | --- |
| 子音組合 | ch, sh, th, sch, ck, ng |
| 母音組合 | ee, ea, ai, ay, oa, ow, ou, oo, oi/oy |
| r-controlled vowels | ar, er, ir/ur, or |
| 長母音規則 | magic e, igh/eigh |
| 子音混合音 | s-blend, l-blend, r-blend |
| 其他常見規則 | soft c, soft g, final y, -le, all |
